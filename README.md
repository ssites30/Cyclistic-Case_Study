# Cyclistic Case Study: How Does a Bike-Share Company Navigate Speedy Success

This project analyzes rider behavior for Cyclistic, a fictional bike-share company in Chicago, to identify strategies for converting casual riders into annual members. The analysis uses real-world Divvy trip data from Q1 2019 and Q1 2020.

# Report Link
[View the HTML Report](https://ssites30.github.io/Cyclistic-Case_Study/)  
[Public Data Source](https://divvy-tripdata.s3.amazonaws.com/index.html)

# Project Objectives
- Understand behavioral differences between casual and annual riders
- Identify patterns in ride duration, station usage, and time-of-day trends
- Recommend data-driven strategies to improve membership conversion

# Data Cleaning Highlights
- Filtered out non-rides and outliers (e.g., test rides, extreme durations)
- Harmonized column names across datasets
- Validated ride records using station logic and duration thresholds

# Tools and Technologies
- R (dplyr, ggplot2, lubridate)
- R Markdown for reproducible reporting
- GitHub Pages for public hosting
